# Fashion-Retail
Fashion Retail Sales Analysis


Company Overview:
Veloura is a premium fashion retail brand offering a wide range of branded apparel for men, women, and kids. Operating across both online and offline stores, Veloura delivers high-quality, stylish clothing across all categories — from casual wear to luxury collections. The company aims to combine fashion and comfort while maintaining strong sales performance across regions.

Project Summary:
 Developed an interactive Power BI dashboard to analyze Veloura’s overall sales, profit, and store performance. The report provides key insights through:

Total Sales and Profit Margin tracking for performance evaluation.
Sales vs Target and Units Sold vs Target Units comparison to measure goal achievement.
Average Order Value (AOV), Year-to-Date (YTD), Previous Year Sales, and Month-over-Month (MoM) growth to assess business trends.
Category-wise and product-level analysis to identify top-performing apparel segments.
Store and region-level performance to pinpoint profitable locations.
Interactive slicers and page navigation for dynamic data exploration.

**There were more than 7 lacs data in the dataset .**

**This dashboard helps Veloura’s management monitor KPIs, assess profitability, and make data-driven decisions for marketing, inventory, and strategic planning.**

Tools Used: **SQL Server (for data cleaning, preprocessing, and extraction), Power BI (for visualization). SQL Server was connected with Power bi for ETL**
